**Theme:** Flexbox deep dive with Tailwind utilities

## Learning Goals

- Use container vs item properties (`flex`, `justify-*`, `items-*`, `gap`).
- Build a responsive row/column switch with `flex-col` -> `md:flex-row`.
- Adjust order with `order-first/last` and add visual differentiation.

## Live‑Coding Demo Checklist

- Three‑card layout using `flex`, `gap`, `items-center`, `justify-center`.
- Add `md:flex-row` (mobile‑first column -> row).
- Demonstrate item order on a second container (`order-first`, `order-last`).

## Common Pitfalls

- Setting width rules that fight Flexbox; forgetting `flex-wrap` when needed.
- Over‑nesting containers when a single flex parent suffices.
