#### Objective

Build a small app that gets user input from the DOM and uses it to call an API with different params.

#### Requirements

1. Create a page with at least two user inputs and a submit button.
2. Use the input values to build an API request with different params.
3. Render loading, error, and success states in the page.
4. Show the returned results in a clear way using `textContent`.
5. README shows how to run the project and what each input changes in the API request.
