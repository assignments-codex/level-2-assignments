**Goal:** Build and style a small form within a semantic layout.

## Steps

1. Start from a semantic shell (`header`, `nav`, `main`, `footer`).
2. Inside `main`, add a form with:
   - Name (text), Email (email), Message (textarea), a Select with at least 3 options, and Submit.
3. Add proper **labels** (`for`/`id`) and use native attributes (`required`, `placeholder`, `type="email"`).
4. Add minimal CSS to align inputs neatly.
5. Use ARIA **only** if needed (e.g., `aria-live="polite"` for inline messages).

**Deliverable (practice):** Push to `w2-d2-form-semantic`.
