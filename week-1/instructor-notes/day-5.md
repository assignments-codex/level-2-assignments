## Learning Goals

- Publish a static page with **GitHub Pages**.
- Use **DevTools breakpoints** (Sources tab, step over/resume).
- Complete and submit the **graded Environment Setup & Git Workflow Readiness**.

## Timeline

### 0–50 min - Demo 1: GitHub Pages + DevTools

- Enable Pages (Settings -> Pages -> Source = **main**).
- Create `cheat-sheet.html` and deploy; visit the URL.
- DevTools: set a breakpoint in a tiny script; step over; inspect scope.

**10‑minute break**

### 60–110 min - Demo 2: Walk the graded checklist

- Walk students through the rubric; clarify what gets submitted and how to make the repo **public**.

### 110–120 min - Breakout

- Students finish submissions; TAs spot‑check repo links in Moodle.
