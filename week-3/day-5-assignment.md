## Objective
Practice arrays using a `for` loop, `forEach`, and `map`, then verify output in DevTools Console.

## Tasks
1. Create `index.html` and `app.js`, link with `<script src="app.js" defer></script>`
2. In `app.js`, create:
   - `const tags = ["js", "week3"]`
   - `const scores = [10, 25, 18, 30]`
3. Use a `for` loop to compute `totalScore` from `scores`, then `console.log(totalScore)`
4. Use `tags.forEach(...)` to log each tag
5. Use `scores.map(...)` to create `doubledScores`, then `console.log(doubledScores)`
6. Open `index.html` with Live Server and confirm logs in DevTools Console

## ACP
Commit after each task section. Final commit message: `build: finish mini-build`

- Setup works (0-5)
- Arrays created (0-5)
- `for` loop total correct (0-5)
- `forEach` and `map` correct (0-5)
