**Goal:** Practice variables/types/operators and object property access (dot vs bracket).

## Steps

1. Create `index.html` and `app.js`; link the script.
2. In `app.js`:
   - Declare `const firstName`, `let score`, `const isActive`, `const tags = ['js','week3']`.
   - Create `const user = { id: 1, 'favorite-color': 'blue' }`.
   - Log `user['favorite-color']` and then update it using a **dynamic key** variable.
   - Write a small function `add(a,b)` that returns a sum; log `add(2,3)`.
3. Open `index.html` and verify output in DevTools Console.
4. Push to a public repo `w3-d1-js-basics`.
