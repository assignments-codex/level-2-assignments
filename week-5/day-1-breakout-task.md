**Goal:** Set up Postman, create a collection and environment, and explore one free API.

## Steps

1. Install or open Postman (web or desktop). Create a collection named `Week5 – Free API` and an environment with variables such as `base_url`.
2. Choose a free API (e.g., JSONPlaceholder or Pokémon). Add a **GET** request using variables, for example: `{{base_url}}/posts` or `{{base_url}}/pokemon/{{name}}`.
3. Send the request and inspect **Status**, **Headers**, and **Body** (JSON). Save the request in the collection.
4. Add one more request to the collection (query params or another endpoint). Use environment variables where helpful.

**Deliverable:** Public repo `w5-d1-postman-basics` with a `README.md` listing the endpoints you tried and one redacted screenshot (optional).
